# Convert Maven Archetype Project to Spring Boot (IntelliJ Guide)

This guide explains how to convert a **Maven Archetype (Quickstart) project** into a **Spring Boot project** using IntelliJ IDEA.

---

# 1. Prerequisites

Make sure the following tools are installed:

* Java **17 or later**
* IntelliJ IDEA
* Maven

Verify installation:

```bash
java -version
mvn -v
```

---

# 2. Create a Maven Starter Project

Open **IntelliJ IDEA**

```
File → New → Project
```

Select:

```
Maven Archetype
```

Choose the archetype:

```
org.apache.maven.archetypes:maven-archetype-quickstart
```

Fill the project details:

```
Name: mavenArchetypeToSpringBoot
Location: your_project_location
GroupId: com.pushan
ArtifactId: mavenArchetypeToSpringBoot
Version: 1.0-SNAPSHOT
```

Click **Create**.

---

# 3. Project Structure After Creation

The project will look like this:

```
mavenArchetypeToSpringBoot
│
├── pom.xml
└── src
    ├── main
    │   └── java
    │       └── com.pushan
    │           └── App.java
    │
    └── test
        └── java
            └── com.pushan
                └── AppTest.java
```

This is a **plain Maven project**, not Spring Boot yet.

---

# 4. Convert pom.xml to Spring Boot

Open `pom.xml`.

Add **Spring Boot parent** after `<modelVersion>`.

```xml
<parent>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-parent</artifactId>
    <version>3.3.2</version>
</parent>
```

This parent acts as the **BOM (Bill of Materials)** and manages dependency versions automatically.

---

# 5. Add Spring Boot Dependencies

Inside `<dependencies>` add:

```xml
<dependencies>

    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-web</artifactId>
    </dependency>

</dependencies>
```

This dependency provides:

* Spring MVC
* Embedded Tomcat
* REST API support
* JSON support

---

# 6. Add Spring Boot Maven Plugin

Add this section to the bottom of `pom.xml`.

```xml
<build>
    <plugins>

        <plugin>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-maven-plugin</artifactId>
        </plugin>

    </plugins>
</build>
```

This plugin allows running the application using Maven.

---

# 7. Reload Maven Dependencies

In IntelliJ:

```
Right Click Project
 → Maven
 → Reload Project
```

Or use the **Reload icon** in the Maven panel.

This downloads all Spring Boot dependencies.

---

# 8. Delete the Default App.java

Delete the file generated by Maven:

```
src/main/java/com/pushan/App.java
```

---

# 9. Create Spring Boot Main Class

Create a new class:

```
DemoApplication.java
```

Location:

```
src/main/java/com/pushan/
```

Code:

```java
package com.pushan;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class DemoApplication {

    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }

}
```

Explanation:

`@SpringBootApplication` enables:

* Auto Configuration
* Component Scanning
* Spring Configuration

---

# 10. Run the Application

Run the main class:

```
DemoApplication.java
```

You should see output like:

```
Tomcat started on port 8080
Started DemoApplication
```

---

# 11. Create Your First REST Controller

Create a package:

```
controller
```

Create file:

```
HelloController.java
```

Code:

```java
package com.pushan.controller;

import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class HelloController {

    @GetMapping("/")
    public String hello() {
        return "Spring Boot Running Successfully!";
    }

}
```

---

# 12. Test the Application

Open a browser:

```
http://localhost:8080
```

Response:

```
Spring Boot Running Successfully!
```

---

# 13. Final Project Structure

```
mavenArchetypeToSpringBoot
│
├── pom.xml
│
└── src
    └── main
        └── java
            └── com.pushan
                │
                ├── DemoApplication.java
                │
                └── controller
                    └── HelloController.java
```

---

# 14. Build the Project

Run the following command:

```bash
mvn clean package
```

The JAR file will be created inside:

```
target/
```

Run the application:

```bash
java -jar target/mavenArchetypeToSpringBoot.jar
```

---

# 15. Recommended Java Version

Spring Boot works best with:

```
Java 17
or
Java 21
```

---

# Conclusion

You successfully converted a **Maven Archetype Quickstart project** into a **Spring Boot application** and created a basic REST API.
